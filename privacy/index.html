Privacy Policy

Last Updated: March 2026

1. Educational Purpose

This website is a student project created for the Noroff Front-End Development program. It uses a public API provided by Noroff for learning purposes only and is not intended for commercial use.

2. User Accounts

Users must register using a valid Noroff student email address. Passwords are not stored by this website directly and are securely handled through the Noroff API authentication system.

3. Local Storage

This website stores your accessToken in your browser's LocalStorage to keep you logged in while browsing the auction listings. You may remove this information at any time by logging out or clearing your browser storage.
