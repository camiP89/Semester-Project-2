Terms of Service

Last Updated: March 2026

1. Educational Project

Student Auction House is a student project developed as part of the Noroff Front-End Development program. The website is created for educational purposes and uses the Noroff Auction API to simulate an online auction platform.

2. User Accounts

To use the auction features, users must register with a valid Noroff student email address. Users are responsible for maintaining the confidentiality of their account information and for all activity that occurs under their account.

3. Auction Listings and Bidding

Users may create listings and place bids using virtual credits provided by the Noroff API. All auctions and bids exist solely within this educational environment and have no real-world value.

4. Acceptable Use

Users agree not to misuse the platform, including posting harmful content, attempting to exploit the API, or interfering with the functionality of the website.

5. Changes to the Service

Because this is a student project, features and functionality may change at any time during development. The project may also be removed or updated without notice.
